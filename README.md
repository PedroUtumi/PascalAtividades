# 📘 PascalAtividades – Algoritmos I

Este repositório reúne atividades desenvolvidas na disciplina de **Algoritmos I**, utilizando a linguagem **Pascal**. O objetivo principal é praticar os conceitos fundamentais de lógica de programação e estruturação de algoritmos.

---

## 🎯 Objetivos

* Desenvolver o raciocínio lógico
* Aplicar conceitos básicos de programação
* Compreender estruturas fundamentais da linguagem Pascal
* Resolver problemas computacionais de forma estruturada

---

## 🧠 Conteúdos Abordados

As atividades incluem exercícios sobre:

* Entrada e saída de dados (`read`, `readln`, `write`, `writeln`)
* Estruturas condicionais (`if`, `case`)
* Estruturas de repetição (`for`, `while`, `repeat`)
* Variáveis e tipos de dados
* Operadores aritméticos e lógicos
* Vetores e matrizes
* Procedimentos e funções

---

## 💡 Observações

* Os códigos foram desenvolvidos com foco didático
* Alguns exercícios podem possuir mais de uma solução válida
* Comentários foram adicionados para facilitar o entendimento
